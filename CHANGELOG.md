## [1.0.0] - 2026-06-29

### Added
- PandaCineForge skill engine (7053-line monolithic SKILL.md)
- Extract script (`extract.py`) for one-click deployment
- Quick Start example (`examples/quickstart.py`)
- Visual assets: logo, hero banner, 2 architecture diagrams
- Bilingual README (English + Chinese)
- AUTHORS.md with founder profile (CN/EN)
- MCP Server adapter (`mcp-server/index.py`)
- Claude Code plugin manifest (`.claude-plugin/plugin.json`)
- OpenClaw skill manifest (`.openclaw/skill.json`)
- ROADMAP.md for 3-year project planning
- Community governance: CONTRIBUTING, CODE_OF_CONDUCT, SECURITY
- Issue templates: bug report, feature request, custom
- Pull request template
- Marketing toolkit: docs/MARKETING.md, docs/OPEN_SOURCE_CHECKLIST.md
- 20 GitHub topics for maximum SEO discoverability

### Features
- 6-layer skill architecture (L0-L5) with cold-start bootstrapping
- 5-tier recall system (R0-R5) for skill memory management
- Multi-lane working memory for parallel skill processing
- 16 high-level director skills + 100+ specialized cinematography skills
- Structured output: YAML frontmatter + machine-parseable Markdown
- Zero-configuration: single file, no dependencies, no API keys

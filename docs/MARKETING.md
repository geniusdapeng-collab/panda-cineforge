## 6. Star-growth playbook (tactics, not spam)

1. **Launch day**: post to X, 微博, 即刻, V2EX, HN, Reddit simultaneously. Pin a repo link.
2. **The first 100 stars matter most** — ask your network personally (DMs), don't mass-blast.
3. **Visuals win on GitHub**: the hero banner + 2 architecture diagrams in the README dramatically increase star conversion. (We have them in `/assets`.)
4. **Tag releases**: every meaningful update = a GitHub Release with changelog. Releases show up in followers' feeds.
5. **Respond fast**: every issue/PR within 24h in the first 2 weeks. Momentum compounds.
6. **Write one deep-dive article** ("How we built an AI-to-AI skill engine for filmmaking") and link the repo. Articles convert 5–10× better than raw links.
7. **Keep the "first" claim visible**: "world's first skill engine for AI Agents in filmmaking" is the differentiator. Repeat it in README hero, release notes, every post.
8. **Cross-language reach**: EN README for global stars + CN README for the massive Chinese AI-builders community. Both ship in this repo.
